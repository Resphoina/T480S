## 02.技巧
+ 加粗任务概要`****`
+ 不可完成`:`——添加冒号`:`
+ 不可完成`星号+空格`——添加星号`*`
+ 首页设置
    + 过滤器名称:首页
    + 过滤器指令:`#收件箱,过期|今天,@进行`
        + 项目筛选
        + 时间筛选
        + 标签筛选
    + 完成后循环`每!周`
+ 颜色管理
    + 每个分类统一颜色
+ **_细致化每个动作_**
    + 利于增加Karma
    + 利于检索
    + 利于查看

+ 首页

+ 今日目标→收藏|今日目标:`过滤器指令:today & !p4`
